Custom maps installation:
-------------------------
Install .VPK add-ons by placing them in this folder.

View and manage add-ons from the main menu: Options->Add-ons.

See this page for creating your own addons:

http://developer.valvesoftware.com/wiki/Swarm_Campaign_Add-on_Tutorial



Important Note:
---------------
'sound.vpk' or 'city17_as.vpk' provides the missing queen sound effects.
So that at least one sound.vpk or city17_as.vpk in addons folder is recommended.
